= Vertical Scaling

Vertical scaling, or "scaling up," boils down to adding more resources to a single server to increase its capacity. This is often the
simplest way to handle increased load, as it doesn't require changes to the application's architecture.

Scaling up can be achieved in several ways. The most direct method is upgrading the server's hardware. In TypeDB Cloud this can be handled
by migrating the cluster to a larger machine size.
