= Select operator
:page-aliases: {page-version}@reference::typeql/queries/get.adoc
:test-typeql: linear

The select operator restricts the stream of rows to only contain the variables specified.

== Syntax

[,typeql]
----
select <var> [ , <var> ... ];
----
