= String
:test-typeql: linear

A UTF-8 character string type.

== In expressions

Strings can be concatenated using the `+` operator.

[,typeql]
----
"Hello," + " world!" == "Hello, world!" 
----

== Defining a string valued attribute

[,typeql]
----
#!test[schema]
define
  attribute name value string;
----

== Inserting a string valued attribute

[,typeql]
----
#!test[write]
insert
  $_ isa name "John";
----
