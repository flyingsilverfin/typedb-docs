= `@cascade` annotation

[NOTE]
====
The `@cascade` annotation is a planned feature and not yet available in TypeDB.
Coming soon!
====


The `@cascade` annotation is used
// tag::overview[]
to specify behavior when deleting a relation's role players.
// end::overview[]

== Syntax

.Syntax
[,typeql]
----
<type label> relates <role type label> @cascade;
----

The `@cascade` annotation does not accept any arguments.
