= IDE Plugins

There are some plugins available, written by the TypeDB community. Since they are not currently officially supported, they are not guaranteed to be up-to-date with the latest versions of TypeQL.


== VSCode

Available on the VSCode marketplace: https://marketplace.visualstudio.com/items?itemName=typedb.typeql[TypeQL]

== Jetbrains

Available in the Jetbrains marketplace: https://plugins.jetbrains.com/plugin/17912-typeql[TypeQL]

== Vim

Highlighter available from the https://github.com/typedb-osi/typeql.vim[Github repository].

== Emacs

Highlighter available from the https://github.com/typedb-osi/typeql-plugin-emacs[Github repository].
